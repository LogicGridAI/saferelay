Before you start

Please read and accept the following before using SafeRelay.

What SafeRelay does:
  • Intercepts clipboard paste events locally in your browser
  • Redacts sensitive patterns using regex matching
  • Stores real values in a RAM-only vault (destroyed on tab close)
  • Reveals values locally with a green lock badge in AI responses
Important limitations:
  • Pattern-based detection — non-standard formats may not be caught
  • Not a guarantee of 100% sensitive data interception
  • Not a substitute for your organisation's security policy
  • Does not make AI workflows automatically HIPAA or GDPR compliant
  • Only outbound request: license verification to api.safepaste.app
No data is ever transmitted except a hashed device fingerprint and your license key for Pro verification. No clipboard content, browsing data, or personal information is sent to any server.
